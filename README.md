# Forest-Fire-Burned-Area-Prediction
Machine learning project to predict forest fire burned area using an end-to-end Orange workflow and multiple models.


## Tools Used
- Orange Data Mining
- PowerPoint
- Excel

## What I Did

- Built a complete machine learning workflow in Orange.
- Removed outliers and preprocessed the dataset.
- Encoded categorical variables.
- Trained and evaluated multiple models such as kNN and SVM.
- Compared model performance using evaluation metrics.
- Created a research poster summarizing methodology, results, and visualizations.
